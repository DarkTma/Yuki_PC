**Yuki PC** — это аниме-помощница для рабочего стола Windows.  
Она живёт поверх твоего интерфейса, умеет общаться текстом и голосом, запускать приложения, открывать сайты, управлять базовыми функциями ПК, озвучивать ответы и реагировать на происходящее на экране.

## 🌸 Возможности

- 💬 **Общение с AI:** Интеллектуальные диалоги на базе быстрой модели Gemini 2.5 Flash.
- 🎤 **Голосовой ввод:** Режим постоянного прослушивания и активация голосом.
- 🔊 **Озвучка ответов (TTS):** Генерация речи через сервисы Gemini.
- 🎭 **Переключаемые характеры Юки:**
  - 🎀 Милая
  - 🕶 Серьёзная
  - 📚 Учебная
  - 🔪 Яндере
  - 💢 Грубая
- 🖥 **Управление Windows-командами:**
  - Запуск приложений и открытие сайтов (включая YouTube, Google).
  - Создание скриншотов.
  - Управление громкостью системы.
  - Управление питанием (выключение, перезагрузка, сон).
- 👁 **Анализ экрана:** Работает как по прямому запросу, так и в режиме фонового наблюдения с реакциями.
- 🎵 **Встроенный музыкальный плеер:** Для локальных треков.
- 🧩 **Кастомные пути:** Гибкая настройка путей к твоим любимым приложениям.
- 🪄 **Визуал и кастомизация:** Скины, chibi-режим и плавная floating-анимация.
- 🎬 **3D/анимационный режим.**
- ⚙ **GUI-окно настроек** и встроенная система логов.
- 🎙 **Поддержка кастомных голосов:** Интеграция с RVC / Applio через `.pth` модели.

## 🛠 Технологии

Проект собран с использованием мощного стека:

* **Python 3.10+** (Устанавливается автоматически)
* **PyQt5** — создание графического интерфейса
* **Google Gemini API** — ядро интеллекта и генерация ответов
* **SpeechRecognition + PyAudio** — распознавание голоса
* **pygame / winsound** — воспроизведение аудио
* **Pillow / ImageGrab** — захват экрана
* **OpenCV + librosa** — обработка медиа и аудиоданных
* **pycaw / comtypes** — системное управление громкостью Windows

## 🚀 Установка и запуск

Благодаря умному скрипту запуска, установка стала максимально простой и автоматизированной!

**1. Скачай проект:**

```bash
git clone [https://github.com/DarkTma/Yuki_PC.git](https://github.com/DarkTma/Yuki_PC.git)
cd Yuki_PC

```

*(Или просто скачай архив с проектом кнопкой `Code` -> `Download ZIP` и распакуй его).*

**2. Настрой API-ключ:**
В корне проекта создай файл `.env` и добавь туда свой ключ от Gemini (получить его можно в [Google AI Studio](https://aistudio.google.com)):

```env
GEMINI_API_KEY=your_api_key_here

```

**3. Автоматическая установка и запуск:**
Просто запусти файл:

```bash
start.bat

```

Скрипт сделает **всё сам**:

* Скачает и установит нужную версию Python (если её нет в системе).
* Автоматически подтянет и установит все необходимые библиотеки из `requirements.txt`.
* Запустит Юки!

*(Для ручного запуска, если у тебя уже всё настроено, можно использовать `python yuki_char.py`)*

## 🎙 Дополнительно: голосовая покраска (RVC / Applio)

Если хочешь, чтобы Юки говорила через кастомную нейросетевую `.pth` модель:

1. Установи и запусти **Applio**.
2. Укажи путь к нему в настройках Yuki.
3. Помести нужный `.pth` голос в папку `voices/`.
4. Включи **Voice Conversion** в настройках интерфейса Yuki.

## 🧠 Как пользоваться

После запуска Юки появится на рабочем столе как интерактивный персонаж. Через графический интерфейс ты можешь:

* Открыть чат или включить микрофон.
* Управлять музыкой.
* Менять скины, характеры и отслеживать логи.
* Включить анализ экрана и настроить шорткаты приложений.

**Примеры голосовых команд:**

* *"Юки, открой YouTube"*
* *"Юки, включи музыку"*
* *"Юки, открой Discord"*
* *"Юки, сделай скриншот"*
* *"Юки, сделай громче"*
* *"Юки, который час"*
* *"Юки, найди в гугле Python threading"*

## ⚠️ Важные примечания

* Проект ориентирован исключительно на **Windows**.
* Для работы интеллекта **обязателен** активный Gemini API key.
* Для работы микрофона требуются системные библиотеки (устанавливаются автоматически, но могут зависеть от настроек ОС).
* Скорость и качество синтеза речи (TTS) зависят от выбранного режима генерации.
